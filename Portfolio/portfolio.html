Simon

This well known memory game can be played right on this page. Like the physical version, this game will produce a random sequence the player has to follow. The sequence increases by one in each round. This continues until the player succcessfully passes round 20. The player can also activate strict mode where the sequence will reset to 1 whenever the player misses an input.
